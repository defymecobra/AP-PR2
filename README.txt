Опис програми

Ця програма знаходить всі прості числа від 0 до числа N, введеного користувачем, використовуючи асинхронне обчислення на Java.

Основні кроки роботи програми
1. Введення даних: Програма запитує у користувача число N (в діапазоні від 0 до 1000), до якого необхідно знайти прості числа.
2. Поділ на потоки: Програма розділяє діапазон від 0 до N на кілька частин, кожну з яких обробляє окремий потік. Це прискорює обчислення.
3. Обчислення простих чисел: У кожному потоці виконується обчислення простих чисел у виділеному діапазоні. Клас `PrimeTask`, що реалізує інтерфейс `Callable`, перевіряє кожне число в діапазоні, чи є воно простим.
4. Збір результатів: Результати обчислень з кожного потоку збираються в загальний список за допомогою `Future`. Використовується `CopyOnWriteArrayList`, що забезпечує безпечне додавання результатів з різних потоків.
5. Виведення результатів: Після завершення обчислень програма виводить список усіх простих чисел до N та час виконання програми.

Призначення класів і методів
- PrimeNumbersFinder: Основний клас, що організовує потоки та обробляє введення користувача.
- PrimeTask: Клас, що реалізує обчислення простих чисел у певному діапазоні в окремому потоці.
